# 619. Biggest Single Number

**Data:** 14/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, GROUP BY, HAVING, ORDER BY  
**Link:** https://leetcode.com/problems/biggest-single-number/description/?envType=study-plan-v2&envId=top-sql-50

## Descrição
A tabela `MyNumbers` contém uma única coluna:
- `num`

Cada linha representa um número inteiro.

Escreva uma solução SQL para **encontrar o maior número que aparece exatamente uma vez** na tabela.  
Se não existir nenhum número que apareça apenas uma vez, retorne `null`.

## Solução (SQL Server)

```sql
select max(num) as num
from (select num,count(num) as qwe from mynumbers group by num having count(num) = 1) as qwer
