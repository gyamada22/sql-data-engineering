## 1211. Queries Quality and Percentage

**Data:** 11/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, AVG, CASE, ROUND, NULLIF, GROUP BY  
**Link:** [Queries Quality and Percentage - LeetCode](https://leetcode.com/problems/queries-quality-and-percentage/description/?envType=study-plan-v2&envId=top-sql-50)

### Descrição
Calcular a qualidade média (rating/position) e a porcentagem de queries com rating < 3 para cada query_name.

### Solução (SQL Server)
```sql
SELECT 
    query_name,
    ROUND(AVG(rating * 1.0 / NULLIF(position, 0)), 2) AS quality,
    ROUND(
        SUM(CASE WHEN rating < 3 THEN 1 ELSE 0 END) * 100.0 / COUNT(*),
        2
    ) AS poor_query_percentage 
FROM queries
WHERE query_name IS NOT NULL
GROUP BY query_name;
