# 1667. Fix Names in a Table

**Data:** 16/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, String Functions  
**Link:** https://leetcode.com/problems/fix-names-in-a-table/

## Descrição
Escreva uma solução para corrigir os nomes de usuários, deixando a primeira letra em maiúscula e o restante em minúscula.

## Solução (SQL Server)

```sql
SELECT user_id,
       LEFT(UPPER(name), 1) + SUBSTRING(LOWER(name), 2, LEN(name)) AS name
FROM Users
ORDER BY user_id;
