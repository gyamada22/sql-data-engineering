# 1822. Delete Duplicate Emails

**Data:** 16/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, CTE, ROW_NUMBER  
**Link:** https://leetcode.com/problems/delete-duplicate-emails/description/?envType=study-plan-v2&envId=top-sql-50

## Descrição
Escreva uma solução para remover registros duplicados na tabela `Person`, mantendo apenas o registro com menor `id` para cada `email`.

## Solução (SQL Server)

```sql
with qwe as (
    select id, 
           email,
           row_number()over(partition by email order by id) as qwer
    from person
)

delete person
from person join qwe on person.id = qwe.id
where qwer > 1
