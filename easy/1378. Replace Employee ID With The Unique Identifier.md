# 1378. Replace Employee ID With The Unique Identifier

**Data:** 12/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, LEFT JOIN  
**Link:** [Replace Employee ID With The Unique Identifier - LeetCode](https://leetcode.com/problems/replace-employee-id-with-the-unique-identifier/)

## Descrição
Escreva uma solução para mostrar o unique_id de cada funcionário. Se um funcionário não tiver um unique_id, mostre null no lugar. Retorne o resultado em qualquer ordem.

## Solução (SQL Server)

```sql
SELECT unique_id, name
FROM Employees LEFT JOIN EmployeeUNI ON EmployeeUNI.id = Employees.id
