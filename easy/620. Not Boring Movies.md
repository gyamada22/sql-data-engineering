# 620. Not Boring Movies

**Data:** 11/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, WHERE, MOD, ORDER BY  
**Link:** [Not Boring Movies - LeetCode](https://leetcode.com/problems/not-boring-movies/?envType=study-plan-v2&envId=top-sql-50)

## Descrição
Escreva uma solução para reportar os filmes com ID ímpar e descrição não "boring". Ordenar por rating em ordem decrescente.

## Solução (SQL Server)
```sql
SELECT 
    id,
    movie,
    description,
    rating
FROM cinema
WHERE description <> 'boring' 
    AND id % 2 = 1
ORDER BY rating DESC;
