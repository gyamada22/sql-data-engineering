# 2356. Number of Unique Subjects Taught by Each Teacher

**Data:** 14/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, GROUP BY, Aggregate  
**Link:** https://leetcode.com/problems/number-of-unique-subjects-taught-by-each-teacher/description/?envType=study-plan-v2&envId=top-sql-50

## Descrição
Escreva uma solução SQL para calcular o número de **disciplinas únicas** (`subject_id`) ensinadas por cada professor (`teacher_id`).

A tabela `Teacher` contém as colunas:
- `teacher_id`
- `subject_id`
- `dept_id`

Cada linha indica que um professor ensina uma disciplina em um determinado departamento.

O resultado deve conter:
- `teacher_id`
- quantidade de disciplinas únicas ensinadas por ele.

## Solução (SQL Server / MySQL)

```sql
SELECT
    teacher_id,
    COUNT(DISTINCT subject_id) AS cnt
FROM Teacher
GROUP BY teacher_id;
