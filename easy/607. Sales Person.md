# 607. Sales Person

**Data:** 25/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, JOIN, NOT IN, CTE  
**Link:** [Sales Person - LeetCode](https://leetcode.com/problems/sales-person/description/?source=submission-ac)

## Descrição
Encontre o **nome dos vendedores que não fizeram pedidos para a empresa 'RED'**.  

## Solução (SQL Server)

```sql
WITH qwe AS (
    SELECT sales_id
    FROM orders 
    JOIN company ON orders.com_id = company.com_id
    WHERE company.name = 'RED'
)

SELECT name
FROM salesperson 
WHERE sales_id NOT IN (
    SELECT sales_id 
    FROM qwe 
    WHERE sales_id IS NOT NULL
);
