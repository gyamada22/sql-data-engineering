# 1587. Bank Account Summary II

**Data:** 01/01/2026  
**Dificuldade:** Easy  
**Tópicos:** SQL, JOIN, GROUP BY, HAVING  
**Link:** Bank Account Summary II - LeetCode

## Descrição
Write a solution to calculate the **balance of each user account** and return only users whose balance is **greater than 10,000**.

---

## Solução (SQL Server)

```sql
SELECT
    u.name,
    SUM(t.amount) AS balance
FROM users u
JOIN transactions t
    ON u.account = t.account
GROUP BY
    u.account,
    u.name
HAVING
    SUM(t.amount) > 10000;
