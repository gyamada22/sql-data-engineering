# 1741. Find Total Time Spent by Each Employee

**Data:** 29/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, GROUP BY, Aggregation, CTE  
**Link:** Find Total Time Spent by Each Employee - LeetCode

## Descrição
Write a solution to calculate the **total time spent by each employee per day**.

An employee may have **multiple clock-in and clock-out records on the same day**, and all working intervals must be included in the total.

---

## Solução (SQL Server)

```sql
WITH daily_work_time AS (
    SELECT
        emp_id,
        event_day,
        SUM(out_time) AS total_out_time,
        SUM(in_time) AS total_in_time
    FROM employees
    GROUP BY emp_id, event_day
)
SELECT
    event_day AS day,
    emp_id,
    total_out_time - total_in_time AS total_time
FROM daily_work_time;
