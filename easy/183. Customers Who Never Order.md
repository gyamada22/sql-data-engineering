# 183. Customers Who Never Order

**Data:** 23/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, LEFT JOIN, NULL  
**Link:** [Customers Who Never Order - LeetCode](https://leetcode.com/problems/customers-who-never-order/description/?source=submission-ac)

## Descrição
Escreva uma solução para reportar todos os clientes que **nunca fizeram um pedido**, utilizando as tabelas `Customers` e `Orders`.

## Solução (SQL Server)

```sql
SELECT c.name AS Customers
FROM Customers c
LEFT JOIN Orders o
    ON c.id = o.customerId
WHERE o.customerId IS NULL;
