# 511. Game Play Analysis I

**Data:** 24/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, Window Functions, DENSE_RANK  
**Link:** [Game Play Analysis I - LeetCode](https://leetcode.com/problems/game-play-analysis-i/description/?source=submission-ac)

## Descrição
Escreva uma solução para reportar o **primeiro login** (`event_date`) de cada jogador (`player_id`) na tabela `Activity`.

## Solução (SQL Server)

```sql
WITH qwe AS (
    SELECT 
        player_id,
        event_date,
        DENSE_RANK() OVER(PARTITION BY player_id ORDER BY event_date) AS dr
    FROM activity
)
SELECT 
    player_id,
    event_date AS first_login
FROM qwe
