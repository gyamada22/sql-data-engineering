# 1174. Immediate Food Delivery II

**Data:** 18/12/2025  
**Dificuldade:** Medium  
**Tópicos:** SQL, CTE, ROW_NUMBER, COUNT, ROUND  
**Link:** https://leetcode.com/problems/immediate-food-delivery-ii/description/?envType=study-plan-v2&envId=top-sql-50

## Descrição
Escreva uma solução para calcular a porcentagem de pedidos de clientes que foram entregues exatamente na data preferida, considerando apenas o **primeiro pedido de cada cliente**. Retorne o valor arredondado para 2 casas decimais.

## Solução (SQL Server)

```sql
WITH cte1 AS (
    SELECT
        order_date,
        customer_id,
        customer_pref_delivery_date,
        ROW_NUMBER() OVER(PARTITION BY customer_id ORDER BY order_date) AS qwe
    FROM delivery
)
SELECT 
    ROUND(
        SUM(CASE WHEN order_date = customer_pref_delivery_date THEN 1 ELSE 0 END) * 100.0 / 
        NULLIF(COUNT(customer_id), 0),
        2
    ) AS immediate_percentage
FROM cte1
WHERE qwe = 1;
